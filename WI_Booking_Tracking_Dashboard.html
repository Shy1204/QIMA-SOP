Work Instruction — Level 3

Booking Tracking Dashboard — Daily Update Process (EMEA)

Phase 2 — Order Booking & Announcement Coordination Team Daily · Mon–Fri
Where this fits: This WI sits within Phase 2 — Order Booking & Announcement. It is the daily routine that keeps the shared coordination dashboard current, ensures new bookings are visible to all coordinators, and carries over remarks from the previous day so no follow-up history is lost.
1
Open IRP View Audit and set filters
a
Go to QIMA Dashboard → IRP View Audit tab
Open the QIMA Dashboard and click the IRP View Audit tab at the top. This is the main data source for all EMEA coordination tracking. Set the Insp Date Range to Today → 360 days. Always click Refresh after adjusting to load real-time data.
b
Verify all EMEA countries are selected in the Factory Country filter
In the Factory Country filter, confirm all EMEA countries are ticked. They are usually pre-selected but always verify — an unticked country means those orders will be missing from the export.

1(b) Remove unnecessary columns

Delete columns that are not part of the shared dashboard before processing the file.

Factory Country filter example. Confirm all relevant EMEA countries are selected before exporting the data.

Export data to Excel
a
Click Export → download and open the file
Click the Export button at the top right of the IRP View Audit page. Download when prompted, then open in Excel.

4
Update the shared team dashboard
a
Open the shared team dashboard Excel file
Open the shared dashboard from the team drive. It has two key tabs: Actual (current data) and Previous (yesterday's data with remarks). The top row contains the action buttons used in the steps below.
Dashboard tabs — Actual is the working view. Previous holds yesterday's data with remarks used by the Vlookup.
b
Click "Move to Previous"
Click Move to Previous. This archives the current Actual data (including remarks) into the Previous tab. This must be done before pasting new data — it preserves the remarks history for the Vlookup in step e.
"Move to Previous" button highlighted. Click this first to archive current data before bringing in new export.
c
Click "Clean Actual"
Click Clean Actual. This clears all rows in the Actual tab (from order number column downwards), leaving the headers ready for fresh data.
"Clean Actual" button highlighted. Clears all order rows in the Actual tab — headers remain intact.
d
Paste the colour-coded export data into the Actual tab
Copy all rows from the cleaned and colour-coded Excel export → paste into the Actual tab starting from the first data row (below headers).
Paste values only — use Paste Special → Values (Ctrl+Alt+V → V) to avoid breaking any existing formulas in the dashboard columns.
Paste the colour-coded data into the Actual tab — data populates from the first row below the headers.
e
Click "Refresh Vlookup"
Click Refresh Vlookup. This automatically matches each order number in the Actual tab against the Previous tab and pulls across any existing remarks. Coordinators' previous follow-up notes reappear against the relevant orders automatically.
5
Identify and assign new bookings
a
Filter the Coordinator column for "NA"
Filter the Coordinator column for NA (blank / not yet assigned). Any row showing NA is a new booking that arrived since the last update. All existing orders already have coordinator initials — only new ones show NA.
After Refresh Vlookup — rows showing #N/A in the Remarks and Coordinator columns are new unassigned bookings to action.
b
Assign coordinator initials to new bookings
Distribute new bookings across coordinators and enter their initials in the Coordinator column. Once initials are added, the coordinator knows they have a new order and should send the first coordination email.
c
Coordinator updates the Remark column after sending the first email
After sending the first coordination email, the coordinator updates the Remark column in the dashboard. This remark carries forward into the next day's update via Vlookup, giving full visibility of follow-up history per order.
Tracking tip: Filter the Remark column by coordinator initials to see all their orders at once — including last email sent date, last reminder, and current status. This is how workload and follow-up compliance can be monitored daily.
